# Android — AiStroyka Manager & Worker

Two app modules and a shared library.

- **AiStroykaManager** — `:AiStroykaManager` — Manager app (dashboard, projects, tasks, reports, team, AI).
- **AiStroykaWorker** — `:AiStroykaWorker` — Worker app (tasks, report, photo, offline sync).
- **shared** — `:shared` — Shared Kotlin/Android code (config, API client, DTOs).

Open the `android` folder in Android Studio. If Gradle wrapper is missing, run `gradle wrapper` from the project root or let Android Studio sync and download it.

## To get npx vercel build green (operator steps)
1. Run: vercel login
2. From repo root: npx vercel link (choose existing project or create)
3. npx vercel pull --yes
4. cd apps/web && npx vercel build

Vercel production deploy uses same pipeline: Root Directory = apps/web, Build Command = npm run build (next build). So npm run build success is the critical gate; vercel build locally replicates that with Vercel env.
